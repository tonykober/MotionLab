Foot Locker — 深度調研報告

工具概述

Foot Locker 是一款免費的 Blender 小型外掛，由開發者 Fatih Pehlevan 開發。其功能非常專一：鎖定選取骨骼在指定範圍內的移動，主要用於修正動畫中腳步滑動（foot sliding）的問題。

解決的問題：

  • Mocap / AI 生成動畫中常見的腳步滑動問題
  • 腳部應該著地靜止時卻仍有微小位移
  • 需要將骨骼移動限制在可接受的閾值範圍內

功能詳細說明

核心功能

功能說明
Lock Bone Movement鎖定選取骨骼的移動在指定範圍內
Range Threshold設定允許的最大移動範圍（閾值）
Frame Range指定需要鎖定的幀範圍
Per-Bone 操作可針對不同骨骼分別設定

工作原理

Foot Locker 的方法相對簡單：

  1. 選取需要鎖定的骨骼（通常是腳部 IK 骨骼）
  2. 設定移動閾值（低於此閾值的移動被消除）
  3. 指定幀範圍
  4. 外掛將該範圍內的微小位移清零，實現「鎖定」效果

使用流程

  1. 進入 Pose Mode
  2. 選取要鎖定的骨骼（如 foot_ik.L / foot_ik.R）
  3. 開啟 N-Panel → Foot Locker tab
  4. 設定鎖定參數（範圍、閾值）
  5. 執行鎖定

安裝/使用流程

安裝

  1. 從 Gumroad 免費下載：https://fatihpehlevan.gumroad.com/l/footlocker
  2. Blender → Edit → Preferences → Add-ons → Install → 選取 zip
  3. 啟用 Foot Locker

使用

  1. 打開動畫好的角色 → Pose Mode
  2. 選取腳部骨骼
  3. N-Panel → Foot Locker
  4. 設定閾值與幀範圍
  5. 執行鎖定 → 腳步滑動消除

優缺點分析

✅ 優點

  1. 完全免費 — 零成本解決腳步滑動
  2. 極度簡單 — 功能專一，學習成本近乎為零
  3. 輕量級 — 小型外掛，不影響其他工作流
  4. 即時見效 — 設定好直接執行，結果立刻可見
  5. 同開發者生態 — 配合 In Placer 形成完整的 mocap 後處理工具組

❌ 缺點

  1. 功能過於簡單 — 只有基本的閾值鎖定，無智慧型接地偵測
  2. 無自動偵測 — 不會自動判斷哪些幀是接地狀態
  3. 無地面適配 — 不處理不平地面或斜坡
  4. 無 IK 補償 — 鎖定腳部後不會自動調整腿部 IK 鏈
  5. 需手動指定 — 每段動畫都需手動設定鎖定範圍
  6. 無批次處理 — 不適合大量動畫自動處理
  7. 文件/教學極少 — 簡單工具無需太多說明，但也缺乏進階用法

建議使用場景

適合作為 retarget 後的快速手動修復工具，用於少量需要精修的動畫：


Kimodo BVH → Retarget（ARP/BVH Retargeter）
    → Foot Locker（手動修腳步滑動）← 本工具
    → Export

核心價值：

  • 零成本、零學習門檻的最後一道手動修復
  • 適合少量動作的快速處理
  • 與 In Placer 搭配使用效果更好

限制：

  • 不適合大量動作的自動化 pipeline
  • 品質不如 ARP IK retarget 或 Foot Flow 的專業方案
  • 需要動畫師手動判斷哪些幀需要鎖定
