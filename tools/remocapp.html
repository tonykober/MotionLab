ReMoCapp

📺 影片展示

AI 即時無標記動捕軟體，用 2-8 個網路攝影機即時捕捉全身+臉部動作。

基本資訊

開發者ReMoCapp
平台Windows
價格付費軟體
官網remocapp.com

功能特色

  • 即時動作捕捉（低延遲）
  • 2-8 攝影機多角度
  • 全身 + 臉部同時捕捉
  • 即時串流至 Blender / UE / Unity
  • BVH / FBX 檔案匯出
  • 無需穿戴標記

輸出格式

即時串流到 DCC 軟體 + BVH / FBX 檔案匯出。雙輸出模式。

優缺點

優點缺點
即時低延遲付費軟體
全身+臉部需多攝影機硬體
Blender/UE 直連Windows only
多攝影機高精度需空間設置

與 Kimodo 整合

可作為即時動捕的精準方案。產出 BVH 直接進入 Kimodo 後處理 pipeline，格式完全相容。骨架定義可能需要 retarget，但 BVH 輸出無需額外轉換。

量產工作流定位

即時高精度動捕方案——全身+臉部同時捕捉，BVH 直出，適合需要互動回饋的動作錄製場景。
