一、工具概述

Webcam Motion Capture 是 VTuber 專用追蹤軟體，使用標準 Webcam 追蹤臉部表情（含 Perfect Sync/ARKit 52 表情）、眼球、嘴唇、上半身和手指。2021 年發佈。

核心能力一句話：Webcam → 臉部+上半身+手指即時追蹤 → 串流到 VTuber 軟體。

解決的問題：VTuber 需要精細臉部追蹤但不想買 iPhone/專用硬體。

四、使用流程

  1. 下載安裝 → 桌面應用（Windows/macOS）
  2. 連接 Webcam → 標準 USB webcam
  3. 校準 → 快速校準追蹤
  4. 選擇追蹤 → 臉部/上半身/手
  5. 連接 VTuber 軟體 → VSeeFace / VTube Studio 等
  6. 即時表演 → 追蹤資料即時串流

六、優缺點分析

✅ 優點

  1. Perfect Sync 52 表情 — 精細臉部追蹤
  2. 手指追蹤 — 少見的 webcam 手指追蹤
  3. 低硬體需求 — 標準 webcam 即可
  4. 可連 iPhone — Face ID 高精度

❌ 缺點

  1. 僅上半身 — 無全身動捕
  2. 不輸出 FBX/BVH — 僅即時串流
  3. VTuber 專用 — 非遊戲動畫工具
