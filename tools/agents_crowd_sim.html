Agents（Complex Crowd Simulation）— 深度調研報告

工具概述

Agents 是一個完全以 Geometry Nodes 構建的 Blender 群眾模擬外掛，由獨立開發者 globglob3D 開發。它是第一個在 Geometry Nodes 中實現真正代理式模擬的工具，包含動畫狀態系統、個體路徑規劃和代理間碰撞迴避。目前處於 Beta 階段。

解決的問題：

  • 大多數 Blender 群眾工具只是「場景填充器」（放置靜態/循環動畫角色），缺乏真正的 AI 邏輯
  • 需要類似遊戲 AI 的複雜群眾行為，但不想離開 Blender 環境
  • 利用 Geometry Nodes 原生效能優勢處理大量代理

安裝/使用流程

安裝

  1. 從 Gumroad 購買下載 zip
  2. Blender → Edit → Preferences → Add-ons → Install
  3. 啟用 Agents 外掛
  4. 需要 Blender 4.3+

使用

  • 透過 Geometry Nodes Modifier 面板設定參數
  • 目前無正式文件，靠 Modifier 中的參數說明學習
  • Beta 版，部分功能尚在開發中

優缺點分析

優點

  • ✅ 極便宜：€18，對比 CrowdSim3D 的 €150-200 非常實惠
  • ✅ Geometry Nodes 原生：利用 Blender 效能優化，可能比 Python 計算更快
  • ✅ 真正代理式：不只是擺放，有 AI 邏輯和碰撞迴避
  • ✅ Blender 4.3+ 現代架構：利用最新 Geometry Nodes 功能

缺點

  • ❌ Beta 階段：功能不完整，可能有 bug
  • ❌ 無文件：沒有教學文件，只有 Modifier 參數說明
  • ❌ 功能較 CrowdSim3D 少：無交通模擬、無節點式行為編輯器
  • ❌ 社群小：一人開發，社群支援有限
  • ❌ Geometry Nodes 限制：受限於 GN 本身的能力

與現有工具的整合可能性

與 Kimodo 的整合

  • Kimodo 產出的動畫可作為 Agents 的動畫狀態資產
  • AI 生成的 walk/run/idle 動作直接映射到代理狀態機

與 Pipeline 的整合

  • Geometry Nodes 結果可烘焙為 mesh/instance 動畫
  • 適合快速原型和小規模群眾場景

對「AI 動作量產工作流」的貢獻定位

定位：輕量級群眾行為原型工具

  • 適合小團隊快速驗證群眾行為概念
  • 低成本入門群眾模擬
  • 未來若功能完善，可成為 CrowdSim3D 的輕量替代

限制： Beta 階段不建議用於正式產出，建議作為評估/原型用途。
