TDPT (ThreeD Pose Tracker) — 深度調研報告

1. 工具概述

TDPT（ThreeD Pose Tracker）是由日本 Digital Standard 公司開發的即時 3D 姿態追蹤系統。透過普通攝影機即時偵測 24 個身體關鍵點並輸出 3D 座標，讓使用者無需專業動捕設備即可進行基礎動作捕捉。

核心能力一句話：用一台攝影機即時追蹤全身 3D 姿態，透過 VMC Protocol 即時串流到 Blender/Unity。 解決的問題：
  • 專業動捕設備昂貴（數十萬台幣起）
  • VTuber/獨立開發者需要低成本即時動捕方案
  • 單一攝影機 3D 姿態估計的即時性需求
  • 動捕資料需即時傳入 DCC 軟體

4. 安裝/使用流程

Windows 版

  • 從 BOOTH 下載免費 Windows 版本
  • 解壓到任意資料夾
  • 連接攝影機或準備影片檔案
  • 啟動 TDPT.exe
  • 選擇攝影機來源或影片檔案
  • 即時追蹤開始，可在視窗中看到 3D 骨架
  • 啟用 VMC Protocol 輸出（Settings → VMC → Enable）
  • 在 Blender 安裝 VMC4B 外掛接收資料

iOS 版

  • 從 App Store 下載 TDPT（免費）
  • 開啟 App，允許攝影機權限
  • 將全身拍入畫面即開始追蹤
  • Premium 功能需一次性內購解鎖（約 650 JPY）
  • 啟用 VMC Protocol 輸出（需同一 Wi-Fi 網路）

    6. 優缺點分析

    優點

    • ✅ 零成本入門：Windows 版免費，iOS Premium 僅 650 日圓
    • ✅ 即時追蹤：低延遲即時輸出，適合直播和互動
    • ✅ VMC Protocol 生態：與 VSeeFace、VMagicMirror 等整合良好
    • ✅ 硬體需求極低：一台攝影機或手機即可使用
    • ✅ 多平台支援：Windows/iOS/macOS/Unity
    • ✅ Unity 開源版本：可嵌入自有專案

    缺點

    • ❌ 僅 24 關鍵點：缺乏手指和臉部細節追蹤
    • ❌ 單攝影機限制：遮擋時精度大幅下降
    • ❌ 無 BVH/FBX 直接匯出：需透過 VMC Protocol 中繼
    • ❌ 精度不如多攝影機方案：適合預視/VTuber，不適合精密動捕
    • ❌ 日文文件為主：英文文件較少
    • ❌ 無批次處理能力：一次只能處理一個來源

    7. 與 Kimodo 的整合可能性

    比較項目TDPTKimodo
    輸入方式攝影機/影片文字 prompt
    輸出格式VMC Protocol（即時）BVH 檔案
    精度中低（24 joints）中高（AI 生成）
    即時性✅ 即時❌ 離線生成
    手指/臉部❌ 無❌ 無
    適用場景參考動作錄製量產動作

    推薦整合方式

    [TDPT 即時動捕] → [快速錄製參考動作]
→ [人工描述動作] → [Kimodo 文字生成高品質版]
→ [ARP Retarget] → [引擎]

    TDPT 的最佳角色是快速參考錄製工具——用來錄製大致動作作為 Kimodo prompt 的視覺參考，或直接用於低品質需求（如 VTuber 驅動）。

    8. 對 AI 動作量產工作流的貢獻定位

    定位：低成本即時預視 + 動作參考錄製器

    [動作設計構思] → [TDPT 快速即時錄製] → [動作參考]
                                              ↓
                                    [轉為 Kimodo Prompt]
                                              ↓
                                    [Kimodo 高品質生成]
                                              ↓
                                    [ARP → 遊戲引擎]
    量產流程中的角色：
    • 不是主力動作生成工具
    • 是「動作構思→文字描述」的視覺化輔助
    • 適合快速驗證動作概念（"這個動作大概長這樣"）
    • VTuber/預視/互動原型的即時動捕方案

    10. 行動建議

    • 下載試用 Windows 版：免費，立即可用，評估追蹤品質是否符合預視需求
    • 測試 VMC → Blender 串流：安裝 VMC4B 確認即時串流穩定性
    • 定位為動作參考工具：不替代 Kimodo，而是輔助 prompt 構思
    • 評估 iOS 版 Premium：如需行動端錄製，650 JPY 解鎖非常划算
    • 探索 Unity Barracuda 版：若有即時遊戲內動捕需求可直接嵌入
