Dollars MoCap 是深圳 SunnyView Tech 開發的低成本動捕產品線，覆蓋從 Webcam AI 動捕到 VR 追蹤器動捕、臉部追蹤等多種方案。主打「傳統 Mocap 1% 成本，80% 品質」。

核心能力一句話：$119-$399 一次買斷 → 永久使用 → BVH/FBX/BIP 匯出 → 本地離線。

解決的問題：

  • 訂閱制動捕長期成本高
  • 需要多種追蹤方案（身/臉/手/VR）但預算有限
  • 中小團隊需要自有動捕能力

六、優缺點分析

✅ 優點

  1. 一次買斷永久使用 — 無訂閱費用
  2. BVH 直接匯出 — 與 Kimodo 格式完全相容
  3. 多產品覆蓋 — webcam/VR tracker/臉/手 都有
  4. 極低成本 — $119-$139 永久
  5. VIVA 品質較高 — VR 追蹤器比純 AI 更穩定
  6. EGAO 臉部免費 — 零成本臉部追蹤

❌ 缺點

  1. 品質聲稱「80%」 — 不如 AAA 級
  2. 中國公司 — 英文文件/支援有限
  3. 無 API/批次 — 手動操作
  4. 軟體 UI 較舊 — 不如新工具美觀
  5. VIVA 需要硬體 — HTC Vive 追蹤器額外成本
  6. Windows Only — 不支援 Mac/Linux
