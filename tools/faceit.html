FaceIt（Facial Expressions & Performance Capture）— 深度調研報告

工具概述

FaceIt 是 Blender 的全方位臉部動畫外掛，集臉部綁定（Rigging）、表情創建、動作捕捉（Motion Capture）和動畫於一體。它能讓任意 3D 角色（寫實人物到卡通角色）快速獲得臉部表情系統，並支援 ARKit、Audio2Face、Hallway Tile 等平台的動捕資料匯入和即時錄製。

解決的問題：

  • 臉部 Shape Keys 創建耗時耗力（52 個 ARKit 表情手動雕刻可能花費數天）
  • 臉部動捕資料匯入 Blender 流程繁瑣
  • 不同拓撲的角色需要重新製作表情，無法複用
  • 即時臉部動捕錄製需要額外軟體

安裝/使用流程

安裝

  1. 從 Superhive（原 Blender Market）購買下載
  2. Blender → Edit → Preferences → Add-ons → Install
  3. 啟用 FaceIt

基本使用流程


1. 準備角色模型（任意拓撲）
2. Setup：標記臉部區域（眼睛、嘴巴、鼻子等）
3. Landmarks：放置臉部特徵點
4. Rigging：自動生成臉部骨架
5. Create Expressions：半自動生成 52 個 ARKit 表情
   - 系統自動生成初始版本
   - 使用者微調每個表情
6. Bake：烘焙為 Shape Keys
7. （選擇性）設定 Control Rig
8. MOCAP：匯入或即時錄製動捕資料

動捕錄製流程（ARKit）


iPhone（Face ID）→ 錄製/串流 ARKit 資料
→ FaceIt Importer 匯入 .csv/.json
→ 自動映射到 52 個 Shape Keys
→ 即時預覽 / 匯出動畫

優缺點分析

優點

  • ✅ 全方位解決方案：從 Rig 到動捕到動畫一站式完成
  • ✅ ARKit 完美整合：52 個 Blend Shapes 自動生成 + 動捕匯入
  • ✅ 任意拓撲：不限於寫實人物，卡通/動物均可
  • ✅ 即時錄製：直接在 Blender Viewport 中 Live 捕捉
  • ✅ 非破壞性：表情可隨時修改、重新烘焙
  • ✅ Control Rig：方便手動微調和動畫
  • ✅ 自訂表情（v2.3）：不限 52 個，可做 Visemes、FACS、舌頭等
  • ✅ 文件完善：ReadTheDocs 官方文件 + 教學影片

缺點

  • ❌ 主要專注 Shape Keys：不做骨骼式臉部 Rig（如 Rigify Face）
  • ❌ 表情品質依賴微調：自動生成的初始表情可能需大量手動修正
  • ❌ ARKit 錄製需 iPhone：Face ID 硬體需求
  • ❌ 非免費：需要購買
  • ❌ 不含 Lip Sync：需搭配 Rhubarb/Parrot 等工具

與現有工具的整合可能性

與 Kimodo 的整合

環節整合方式
身體 + 臉部組合Kimodo 生成身體動作 + FaceIt 提供臉部表情 = 完整角色動畫
AI 動作 + 手動臉部AI 無法生成精細臉部動畫，FaceIt 填補此缺口
動捕全身 + 臉部分離身體用 Kimodo/BVH，臉部用 ARKit + FaceIt

與 Blender → UE/Unity Pipeline 的整合

流程說明
ARKit Shape Keys → UE52 個 ARKit 表情直接對應 UE 的 Morph Target 系統
FBX 匯出FaceIt 的 Shape Keys 可隨角色一起匯出 FBX
UE Live Link Face可搭配 UE Live Link Face App 做即時臉部動捕串流

建議整合流程


角色建模 → FaceIt 臉部 Rig + 表情 → ARP 身體 Rig
→ Kimodo 身體動作 + ARKit 臉部動捕
→ ARP Game Export（含 Shape Keys）→ UE/Unity

對「AI 動作量產工作流」的貢獻定位

定位：AI 動作工作流中的臉部動畫缺口填補者


[AI 身體動作]  +  [FaceIt 臉部動畫]  =  [完整角色表演]
  (Kimodo)         (ARKit Mocap /        (身體+臉部)
                    手動/Audio2Face)

具體貢獻：

  1. 填補 AI 盲區：Kimodo 等 AI 工具只生成身體動作，臉部完全空白，FaceIt 完美補充
  2. ARKit 動捕低成本：只需 iPhone 即可獲得高品質臉部動捕
  3. 批次角色表情：一次建好表情預設，可套用到多個角色
  4. 遊戲引擎直通：52 ARKit Shape Keys 是業界標準，直接對應 UE/Unity
