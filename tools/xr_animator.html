一、工具概述

XR Animator（原 System Animator Online）是瀏覽器端 AI 全身動捕工具，使用 Google MediaPipe + TensorFlow 即時追蹤。支援 VRM 模型驅動，面向 VTuber 和 XR 應用。由獨立開發者 ButzYung 維護。

核心能力一句話：瀏覽器開啟 → Webcam 即時全身/臉/手追蹤 → VRM 角色驅動 → VMC 串流。

解決的問題：

  • VTuber 需要全身動捕但預算有限
  • 安裝動捕軟體門檻高
  • 需要即時表演工具

四、使用流程

  1. 開啟網頁 → https://sao.animetheme.com/XR_Animator.html
  2. 允許攝影機 → 瀏覽器授權 webcam
  3. 載入 VRM → 上傳自訂 VRM 模型（或用內建）
  4. 選擇追蹤 → 勾選臉/身/手的組合
  5. 開始追蹤 → 即時在瀏覽器中驅動角色
  6. 串流 → 用 VMC Protocol 送到 VROOM/VSeeFace 等
  7. 錄製 → 在瀏覽器內錄製表演

六、優缺點分析

✅ 優點

  1. 完全免費零門檻 — 開啟網頁即用
  2. 無需安裝 — 瀏覽器端，跨平台
  3. 全身+臉+手 — 追蹤覆蓋完整
  4. 手機也能用 — 行動端可運行
  5. 開源可修改 — MIT 授權

❌ 缺點

  1. 不輸出 FBX/BVH — 不產生遊戲可用動畫檔案
  2. 精度有限 — MediaPipe 等級，不如商業工具
  3. 主要是即時工具 — 不適合後處理工作流
  4. VTuber 導向 — 非遊戲動畫設計
  5. 無 retargeting — 只驅動 VRM，不能轉到任意骨架

八、對「AI 動作量產工作流」的貢獻定位


XR Animator 定位：「零成本即時動作預覽」

• 快速確認動作可行性（5 秒開啟看看）
• VTuber 即時表演
• 教學展示 AI 動捕概念

不適合量產工作流的任何環節。
