B2B SDK — 讓玩家在遊戲內用影片/文字即時生成自訂動畫（emotes）

核心能力

Kinetix 是一家法國 AI 動畫公司，提供遊戲開發者嵌入式 SDK。玩家可在遊戲中直接上傳影片或輸入文字描述，AI 即時生成自訂 3D emote 動作。定位為 B2B 平台，非個人動畫工具。

功能說明
Video-to-Animation玩家上傳影片 → 3D 動作
Text-to-Animation文字描述 → 3D 動畫
Unity/Unreal SDK原生引擎嵌入
UGC 市場玩家可交易自訂動畫
Kamo-1 影片模型3D 條件化 AI 影片生成
即時預覽遊戲內即時看到效果

定位與商業模式

Kinetix 是 B2B 平台，不是動畫師的日常工具。它服務的是「遊戲上線後」的玩家 UGC 需求。

項目內容
目標客戶遊戲開發工作室
商業模式SaaS API 按呼叫量計費
引擎支援Unity / Unreal / Web
Free Tier有（有限 API 呼叫）
Enterprise客製報價

關鍵區分：

  • 開發期動作量產 → Kimodo（我們用）
  • 營運期玩家 UGC → Kinetix（給玩家用）

優缺點

Kinetix 在遊戲內 AI UGC 動畫領域處於領先地位，但不是我們動作設計的日常工具。

  • 遊戲內原生整合 — 玩家無需離開遊戲
  • Text-to-Animation — 降低創作門檻
  • 多引擎支援 — Unity + Unreal + Web
  • UGC 生態 — 增加遊戲黏性和營收
  • B2B Only — 個人無法直接使用
  • 品質不確定 — 玩家生成品質參差
  • 價格不透明 — 需個案洽談
  • 雲端依賴 — 無離線模式

與我們工作流的關係

Kinetix 和我們的 Kimodo 工作流是「不同階段」的工具——一個用於開發，一個用於營運。

階段工具用途
開發期Kimodo + ARP製作角色預設動作庫
營運期Kinetix SDK玩家自訂 UGC 動畫

兩者可在同一遊戲中共存：Kimodo 做基礎，Kinetix 做擴展。

總結與建議

Kinetix 不在動作設計師的日常工具鏈中，但值得推薦給遊戲企劃/製作人了解。
