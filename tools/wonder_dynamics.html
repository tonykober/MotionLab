Wonder Dynamics (Autodesk Flow Studio) — AI VFX + 動捕平台

> 調研日期：2026-06-19

> 官網：https://wonderdynamics.com

> 現為 Autodesk 旗下（2024 收購，2025 更名 Flow Studio）

一、工具概述

Wonder Dynamics（現 Autodesk Flow Studio）是雲端 AI VFX 平台，自動化 80-90% VFX 流程：從實拍影片提取動作→替換為 CG 角色→匹配燈光→合成。2024 年被 Autodesk 收購，2025 年更名。

核心能力一句話：上傳實拍影片 → AI 自動提取身/臉/手動作 + 替換 CG 角色 + 匹配燈光 + 合成完成。

解決的問題：

  • 傳統 VFX 需要 Mocap 棚 + 綠幕 + 大量手工合成
  • 獨立創作者無法負擔 VFX pipeline
  • 從影片到 CG 角色的流程極其複雜

四、使用流程

  1. 註冊 → https://wonderdynamics.com（或 Autodesk 帳號）
  2. 上傳影片 → 實拍影片（手機/攝影機皆可）
  3. 選擇 CG 角色 → 上傳自己的或用內建
  4. AI 處理 → 自動：動捕+燈光匹配+角色替換+合成
  5. 預覽結果 → 在瀏覽器中預覽
  6. 匯出 →
  • 合成影片（直接使用）
  • MoCap 資料（匯入 Blender/Maya 編輯）
  • 3D 場景

六、優缺點分析

✅ 優點

  1. 一體化 VFX Pipeline — 從影片到完成合成一站式
  2. 身+臉+手全追蹤 — 品質高
  3. Autodesk 支持 — 長期穩定、持續更新
  4. 免費方案 — 可試用
  5. CG 角色替換 — 獨特功能（競品沒有）
  6. 已推出免費方案 — 2025 年大幅降價

❌ 缺點

  1. 核心是 VFX 不是 Mocap — 純動畫量產不是主要用途
  2. Credit 按秒計費 — 量產成本可能高
  3. 雲端依賴 — 不能離線
  4. MoCap 匯出細節不明 — 格式/骨架標準文件較少
  5. 被收購後品牌混亂 — Wonder Studio / Flow Studio 名稱混用

七、Pipeline 整合分析

項目說明
輸入來源實拍影片（任何攝影機）+ CG 角色模型
輸出格式合成影片 / AI MoCap 資料（匯出到 DCC）
骨架標準標準骨架（含臉+身+手）
上游工具攝影機拍攝 + 自訂 CG 角色
下游工具Blender（場景匯出編輯）、Maya、After Effects（VFX 合成）

推薦搭配：Wonder Dynamics (Flow Studio) → AI VFX 合成影片（遊戲預告片/行銷影片）。或用 AI MoCap 模式提取動作 → Blender/Maya。核心用途是 VFX 不是純動畫量產。

八、對「AI 動作量產工作流」的貢獻定位


Flow Studio 定位：「遊戲預告片/行銷影片的 AI VFX 工具」

不是量產動畫的工具，而是：
• 製作遊戲宣傳影片（實拍+CG 角色合成）
• 快速概念影片 pre-viz
• 過場動畫 mockup

量產動畫請用 Viggle / Motorica / Kimodo
