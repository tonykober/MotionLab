Pipeline 整合分析

本工具在 AI 動作量產工作流中的定位與搭配方式。

  • 輸入來源 — 移動參數（速度/加速/轉向）、路徑繪製、風格選擇/混合、Style Clone（10 分鐘 Mocap）
  • 輸出格式 — FBX（標準動畫格式）、UE5 Plugin 直入
  • 骨架標準 — Motion Matching 相容資料集
  • 上游工具 — 自有 Mocap（用於 Style Clone）
  • 下游工具 — UE5（Plugin 自動 Retarget + Pose Search Database）、Unity、Maya、Blender、Godot

量產工作流定位

Motorica 負責遊戲角色的移動層動畫。

Locomotion 層 ← Motorica（走/跑/轉向/NPC行為） 表演動作層 ← AI 動作生成工具 + Viggle（戰鬥/手勢/特效） 混合清理層 ← Blender retarget 匯出 → UE5 / Unity
