一、工具概述

Move.AI 是英國 AI 動捕公司（2019 年成立），開發企業級無標記動作捕捉技術。其旗艦產品 Genesis 是首款聲稱可達「光學系統品質」的 AI 動捕方案，目標客群為大型遊戲/VFX/電影工作室。

核心定位：取代 Vicon/OptiTrack 等傳統光學動捕系統，用 AI + 多鏡頭達到同等品質但更低成本和更簡單設置。

解決的問題：

  • 傳統光學 Mocap 系統動輒 $50,000+
  • 標記點（marker）穿戴繁瑣
  • 需要專門 Mocap 場地
  • 後處理清理工作量大

知名客戶：EA、Ubisoft (Just Dance 2023)、Sonic 3 電影、Shōgun FX 影集、Nike x UEFA、Grimes @ Coachella

四、使用流程

Move One (個人)

  1. 下載 iPhone App
  2. 用 iPhone 拍攝動作影片
  3. 上傳處理
  4. 下載 FBX/BVH/GLB

Move API (開發者)

  1. 註冊：https://portal.move.ai
  2. 上傳影片（單/多鏡頭）
  3. 選擇模型 (s1/s2/m1/m2)
  4. 處理 → 取得動畫資料
  5. 匯出所需格式

Genesis (企業)

  1. 安裝 6-12 台 Z-Cam + Nvidia GPU
  2. 棋盤格校準
  3. 拍攝表演
  4. 本地端 AI 處理（100% on-premise）
  5. 匯出所有格式

六、優缺點分析

✅ 優點

  1. 品質最高：聲稱達光學系統等級，EA/Ubisoft 驗證
  2. 格式最完整：FBX/BVH/GLB/USDZ/USDC/Blend/C3D/JSON 全支援
  3. 有 API：支援批次處理
  4. 手部追蹤：s2/m2 模型內建
  5. 多人追蹤：最多 20+ 人
  6. 本地端部署：Genesis 100% on-premise，無雲端依賴
  7. 即時串流：rt1/rt2 模型支援
  8. 生物力學輸出：關節力矩/地面反力
  9. 頂級客戶背書：EA、Ubisoft、Sony Music
  10. Blender 原生輸出：直接 .blend 檔

❌ 缺點

  1. 定價偏高：Move One 起步 $18/月，Genesis 企業級
  2. iPhone 限定：Move One App 只有 iOS
  3. 無臉部追蹤：目前沒有 face capture
  4. Genesis 硬體成本：需要多台攝影機+GPU
  5. API 為 Beta：尚在開發中
  6. 品質嚴重依賴條件：光線、遮擋、攝影機數量都影響
  7. 非即時（除 Enterprise）：一般用戶都是後處理

七、與 Kimodo BVH 工作流的整合可能性

面向評估
格式相容⭐⭐⭐⭐⭐ BVH 直出
品質層級Move.AI > Kimodo（在有影片參考的場景）
API 批次✅ 可自動化
成本偏高，不適合大量低預算量產
定位「高品質影片 Mocap」vs Kimodo「text-to-motion」
適合場景重要角色/主角動作用 Move.AI 品質，NPC 用 Kimodo 量產

八、對「AI 動作量產工作流」的貢獻定位


Move.AI 在工作流中的角色：
「品質天花板」— 當需要最高品質動捕時的選擇

量產工作流分層：
• 頂級品質（主角/Boss 動作）→ Move.AI Genesis
• 高品質（重要 NPC）→ Move.AI API / Viggle
• 量產級（雜兵/背景 NPC）→ Kimodo + Motorica
