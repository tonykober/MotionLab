Auto-Rig Pro（Retargeting 功能面）— 深度調研報告

工具概述

Auto-Rig Pro 是 Blender 上最知名的付費 Rigging 外掛之一，由開發者 Artell 開發。除了自動化角色綁骨的核心功能外，內建的 Remap 模組 是 Blender 生態中最穩健可靠的 Retargeting 工具之一。

解決的問題（Retargeting 面）：

  • 將 Mocap / AI 生成的動畫從來源骨架映射到目標角色骨架
  • 處理不同比例、不同 Rest Pose（T-Pose vs A-Pose）的骨架間動畫轉移
  • 提供 IK Retargeting 解決腳步滑動問題
  • 多動畫批次 retarget
  • 遊戲引擎 FBX/GLTF 匯出

IK Retargeting（關鍵特色）

ARP 的 Remap 模組能將 FK 動畫轉為 IK 目標骨骼的動畫：

  • 腳步使用 IK retarget → 有效避免滑步（foot sliding）
  • FK retarget → 速度快但會有明顯滑步
  • IK Pole 三種計算模式對應不同品質的輸入動畫

使用流程

  1. 匯入來源動畫（BVH / FBX）
  2. 指定 Source & Target Armature
  3. Auto-Scale 對齊尺寸
  4. Build Bones List → 自動/手動映射
  5. Redefine Rest Pose（如需要）
  6. Re-Target → 動畫轉移完成
  7. Interactive Tweaks（可選微調）

安裝/使用流程

安裝

  1. 購買：Superhive Market (原 Blender Market) 或 Gumroad
  2. 下載 zip
  3. Blender → Edit → Preferences → Add-ons → Install → 選取 zip
  4. 啟用 Auto-Rig Pro

Retargeting 使用流程

  1. 確保場景中有 Target Armature（ARP rigged 或 Rigify 或自訂）
  2. 匯入 Source 動畫（File → Import → BVH/FBX）
  3. 開啟 ARP 的 Remap 面板
  4. 設定 Source / Target
  5. Build Bones List → 選擇或自訂 Preset
  6. 如需要，Redefine Rest Pose
  7. 點擊 Re-Target

優缺點分析

✅ 優點

  1. IK Retargeting — 能有效減少腳步滑動，品質遠超純 FK retarget
  2. 內建 Presets — Mixamo、Rokoko、XSens 等開箱即用
  3. Rest Pose 處理 — 優雅處理 T-Pose / A-Pose 差異
  4. Interactive Tweaks — retarget 後直覺式微調
  5. Multiple Animations — 可批次處理多動畫
  6. In Place — 一鍵原地化循環動畫
  7. 遊戲引擎整合 — 直接匯出 Unity/Unreal/Godot 格式
  8. 完整文件 — 官方文件完善，CGDive 有大量影片教學
  9. 穩定可靠 — Blender 社群公認最穩的 retarget 方案之一
  10. 全流程整合 — Rig + Retarget + Export 一站式完成

❌ 缺點

  1. 付費 — $40 USD 門檻（但性價比高）
  2. 非開源 — 無法自訂內部邏輯
  3. Preset 偏向 ARP Target — 內建預設以 ARP 骨架為目標，其他骨架需手動建映射
  4. 學習曲線 — 功能豐富但初次設定需時
  5. 無 Python API 批次接口 — 官方未提供 headless 批次 retarget 的 Python API（社群有第三方方案）
  6. Interactive Tweaks 限 Euler — 不支援 Quaternion 模式的即時調整
