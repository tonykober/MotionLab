Foot Flow — 深度調研報告

工具概述

Foot Flow 是一款付費的 Blender 外掛，透過 Superhive Market (原 Blender Market) 販售。專門用於清理和穩定 Mocap 動畫中的腳步滑動及其他常見腳部問題。相比免費的 Foot Locker（簡單閾值鎖定），Foot Flow 提供更專業、更自動化的腳步修正方案。

解決的問題：

  • Mocap / AI 生成動畫中的腳步滑動（foot sliding）
  • 腳部不穩定/抖動
  • 接地問題（腳部穿透地面或浮空）
  • 其他常見的腳部動畫瑕疵

安裝/使用流程

安裝

  1. 從 Superhive Market 購買並下載
  2. Blender → Edit → Preferences → Add-ons → Install → 選取 zip
  3. 啟用 Foot Flow

使用流程（推測基於功能描述）

  1. 選取帶有動畫的角色骨架
  2. 指定腳部骨骼
  3. 設定接地參數
  4. 執行腳步修正
  5. 預覽結果並微調

優缺點分析

✅ 優點

  1. 專業 Mocap 清理 — 專為 motion capture 數據設計
  2. 比 Foot Locker 更智慧 — 不是簡單閾值，有更進階的偵測邏輯
  3. 多種腳部問題處理 — 滑動+穩定+接地，不只是鎖定
  4. Blender Market 品質 — 通過市場審核的付費工具
  5. 適合量產 — 比手動方案效率高

❌ 缺點

  1. 付費 — 需要購買，增加成本
  2. 公開資訊有限 — 付費工具詳細功能文件不在公開可見範圍
  3. 社群反饋少 — 相比 ARP 等老牌工具，使用者反饋較少
  4. 可能不如 ARP IK — ARP 的 IK retarget 從源頭解決滑步，Foot Flow 是後期修復
  5. 依賴骨架結構 — 可能對骨架命名/結構有特定要求

    → Foot Flow ← 本工具定位

      - 穩定腳部
    → Export

核心價值：

  • 當 ARP IK retarget 不可用或不足時（如用免費工具鏈 FK retarget）
  • 提供後處理階段的專業腳步修正
  • 比 Foot Locker 更適合量產工作流

與其他腳步工具的定位區分：

工具定位時機
ARP IK Retargetretarget 時直接解決最前端
Foot Flowretarget 後專業修正中間
Foot Locker最後手動微調最末端

購買建議：

  • 如果已有 Auto-Rig Pro（含 IK retarget），Foot Flow 的必要性降低
  • 如果使用免費工具鏈（BVH Retargeter FK），Foot Flow 是重要的品質補充
  • 需要確認 Foot Flow 是否支援批次處理，這對量產工作流至關重要
