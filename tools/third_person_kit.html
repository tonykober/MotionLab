Third Person Kit — 深度調研報告

工具概述

Third Person Kit 是 Blender 外掛，由 rileyb3d 開發，能讓使用者在 Blender 場景中像玩第三人稱遊戲一樣控制角色。透過鍵盤（WASD）操控角色行走、奔跑、蹲下、跳躍，並即時錄製移動軌跡為 Blender 關鍵幀動畫。

解決的問題：

  • 角色移動動畫（locomotion）手動 K 幀耗時
  • 動畫預視（previs）/故事版（animatics）需要快速產出角色移動路徑
  • 從 Mixamo 等來源匯入動畫後，缺乏在場景中「操控」角色的方式
  • 遊戲開發前期需要快速驗證角色移動手感

工作流


角色 FBX（含 Rig）+ 動畫（Idle/Walk/Run/Crouch/Jump）
→ Third Person Kit 設定
→ 按 Play 進入「遊戲模式」
→ WASD 控制角色在場景中移動
→ 退出後動作存為標準 Blender Action
→ 可在 NLA Editor 中編輯/組合

Mixamo 快速工作流

  1. 從 Blender 匯出角色 FBX
  2. 到 Mixamo 取得基礎動作（walk/run/crouch/jump）
  3. 匯回 Blender
  4. 設定 Third Person Kit
  5. 開始操控

安裝/使用流程

安裝

  1. 從 Superhive 購買（$24 USD）
  2. Blender → Edit → Preferences → Add-ons → Install
  3. 啟用 Third Person Kit
  4. 支援 Blender 4.2 - 5.0（v2.1.0）

使用流程


1. 匯入帶 Rig 的角色 FBX
2. 匯入動畫（Walk/Run/Crouch/Jump/Idle）
3. 在 Third Person Kit 面板中設定：
   - 綁定角色
   - 指定各動作 Action
4. 設定場景碰撞（地面、牆壁）
5. 按 Play → 進入控制模式
6. 使用鍵盤/手把操控角色
7. 退出 → 動畫自動存為 Action
8. 在 NLA Editor 中使用/編輯

優缺點分析

優點

  • ✅ 即時操控動畫：像玩遊戲一樣製作移動動畫
  • ✅ 極快速 Previs：數分鐘內完成角色場景移動
  • ✅ Mixamo 整合：快速工作流
  • ✅ 攝影機系統：含碰撞/縮放/Free Look，適合過場動畫
  • ✅ 錄製為標準 Action：可在 NLA 中組合、編輯
  • ✅ Gamepad 支援：手把操控更直觀
  • ✅ 價格合理：$24，功能完整
  • ✅ 碰撞系統：角色不會穿牆

缺點

  • ❌ 不是精細動畫工具：不能取代專業動畫師的細膩動作
  • ❌ 依賴預製動畫：需要先有 walk/run/jump 等動畫
  • ❌ Blender 5.0 相容問題：Action Slot 系統變更導致部分問題
  • ❌ 僅限移動動畫：不處理上半身/手部/臉部動畫
  • ❌ 角色 Rig 需一致：所有 FBX 的骨架需統一

對「AI 動作量產工作流」的貢獻定位

定位：AI 動作的即時編排器 + 場景走位工具


[AI 動作庫] → [Third Person Kit] → [場景走位動畫] → [完整過場/遊戲動畫]
 (Kimodo)      (即時操控決定       (何時走、何時跑    
                路線和時機)          何時跳、何時停)

具體貢獻：

  1. 動作→場景的橋樑：AI 生成的是「動作本身」，TPK 負責「何時在哪裡用」
  2. 極速 Previs：遊戲過場動畫的快速原型驗證
  3. 非動畫師也能用：設計師/企劃可自己操控角色走位，減少溝通成本
  4. 攝影機同時錄製：動作+運鏡一次完成

適用場景：

  • 遊戲過場動畫 previs
  • 場景走位規劃
  • 角色移動動畫快速產出
  • 動畫 Reference 錄製
