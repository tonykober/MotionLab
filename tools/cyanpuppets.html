一、工具概述

CyanPuppets（青瞳視覺 / CYAN.AI）是中國 AI 動捕公司，擁有十億參數動捕大模型。從攝影機/影片即時提取 208 個關鍵點（臉 140 + 手指 21 + 四肢 30），延遲僅 0.1 秒。

核心能力一句話：十億參數 AI → 208 關鍵點（臉140+手21+肢體30）→ 0.1 秒延遲即時 → Blender/UE 插件。

解決的問題：

  • 普通 AI 動捕追蹤點少、精度不足（尤其臉/手）
  • 需要分開用不同工具追蹤臉/身/手
  • 延遲太高無法即時預覽

六、優缺點分析

✅ 優點

  1. 208 追蹤點 — 市場上數一數二的精細度（臉 140 點）
  2. 十億參數大模型 — AI 品質理論上更高
  3. 0.1 秒延遲 — 接近即時
  4. Blender + UE 插件 — 直接整合主流引擎
  5. 臉+身+手一體 — 不需分開工具
  6. 免費版開發中 — 未來可能零成本使用

❌ 缺點

  1. $50/月偏貴 — 比 Marionette($12.50) 和 Viggle($0.03/clip) 貴很多
  2. 中國產品 — 英文文件較少
  3. 市場認知度低 — 獨立品質驗證少
  4. 免費版未發佈 — 目前只能付費
  5. 匯出格式不明確 — 官方文件未詳列 BVH/FBX 等格式支援

八、對「AI 動作量產工作流」的貢獻定位


CyanPuppets 定位：「高精度臉部+手指追蹤方案」

最適合：
• 需要精細臉部表情（140 點）的角色演出
• 需要手指動畫的場景
• 已有 Blender/UE Pipeline

等免費版後：
• 可能成為免費高精度動捕的首選

現階段不適合：
• 預算有限的量產（$50/月 vs Viggle $0.03）
• 只需要全身動捕的場景（overkill）
