QuickMagic

AI 無標記動捕 — 手機/網頁即時

QuickMagic 是一款 AI 驅動的無標記動作捕捉工具，支援手機拍攝與網頁即時動捕兩種模式。只需一支手機或 Webcam 即可完成動作捕捉並匯出 FBX/BVH 格式，提供從免費試用到付費訂閱的彈性方案。

AI 動捕手機拍攝網頁即時FBX/BVHAPI 支援

核心能力

QuickMagic 讓你用手機或 Webcam 即時進行 AI 動作捕捉，支援 FBX/BVH 輸出到 Blender/UE/Unity。提供免費試用和 API 批次處理能力。

⚠️ 傳統 Mocap

  • 專業攝影機 — 需要多攝影機陣列或深度感測器
  • 場地限制 — 只能在特定空間使用
  • 後處理時間 — 拍攝後需要長時間資料清理
  • 高單價 — 每個動畫 $500+ 的製作成本

✅ QuickMagic 方案

  • 手機即可 — iPhone/Android 手機直接當動捕攝影機
  • 任何場地 — 室內外不限，隨時隨地捕捉動作
  • 即時輸出 — BVH/FBX 快速輸出，免長時間後處理
  • API 批次 — 有 API 支援大量動畫自動化處理

工具示範

影片展示 QuickMagic 的手機動捕模式與網頁即時動捕功能，包含從拍攝到匯出完整動畫檔案的全流程演示。可觀察其追蹤精度與處理速度。

技術規格

FBX

標準 3D 動畫匯出格式，直接相容主流引擎

BVH

骨架動作資料格式，支援動作重定向與編輯

API

提供程式化介面，可整合進自動化工作流程

平台與整合

  • Blender — FBX/BVH 匯入後可進行動作編輯與微調
  • Unity — FBX 格式直接支援 Humanoid Retarget 工作流程
  • Unreal Engine — 匯出檔案相容 UE 動畫系統
  • 手機 App — iOS/Android 手機 App 進行拍攝式動捕
  • API 整合 — 可透過 API 將動捕功能嵌入自有 Pipeline

定價方案

免費試用 + 付費

QuickMagic 提供免費試用額度讓使用者評估品質，正式使用則需訂閱付費方案。付費方案依處理量與功能等級分層，包含更高的處理優先權、更多匯出格式選項與 API 存取權限。具體價格可參考官網最新方案。

優缺點分析

✓ 優點

  • 低硬體門檻 — 手機即可完成動捕，不需專業設備
  • 雙模式 — 手機拍攝離線處理 + 網頁即時追蹤兩種模式
  • API 支援 — 可程式化整合，適合自動化流程
  • 免費試用 — 可先評估品質再決定是否付費

✗ 缺點

  • 付費才能正式使用 — 免費額度有限，正式產出需訂閱
  • 品質依拍攝條件 — 光線、角度、遮擋都會影響提取品質
  • 雲端依賴 — 部分功能需雲端處理，有延遲與隱私考量
  • 生態較新 — 社群資源與教學相對主流工具較少

與 Kimodo 整合

QuickMagic 可作為 Kimodo 工作流程中「影片轉 BVH」的替代方案。當需要將實拍參考動作轉化為可編輯的動畫資料時，QuickMagic 提供了從手機拍攝到 BVH 匯出的一站式解決方案，其 API 更可能實現自動化串接。

  • 替代方案 — 作為影片轉 BVH 的備選工具，與 Marionette 互為替代
  • API 自動化 — 可透過 API 自動提交影片、自動下載 BVH 結果
  • 品質比對 — 將 QuickMagic 產出與 Kimodo 生成結果進行品質對照

工作流程定位

QuickMagic 在流程中提供「低成本動作數位化」的能力。團隊成員可以隨時用手機拍攝參考動作，透過 QuickMagic 快速轉為 BVH，然後再決定是否直接使用或轉由 Kimodo 重新生成更高品質版本。

  • 拍攝收集 — 用手機隨時拍攝動作參考素材
  • 快速轉換 — QuickMagic 將影片轉為 BVH/FBX（本工具）
  • 品質決策 — 評估是否直接使用或需要 Kimodo 重新生成

QuickMagic 以手機動捕的便利性和 API 整合能力，成為動作生產流程中靈活的工具選項。特別適合需要大量快速動作數位化的場景，其 API 為自動化流程提供了可能性。

  • 免費試用 — 註冊帳號測試免費額度，評估動捕品質
