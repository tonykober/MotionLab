TDPT

ThreeD Pose Tracker

即時 3D 姿態追蹤工具——僅需一台普通攝影機，即可偵測 24 個身體關鍵點並輸出完整 3D 座標資料。由 Digital Standard 開發，專為即時動作預覽與串流設計。

即時追蹤單攝影機VMC Protocol付費軟體

核心能力

傳統動捕需要昂貴設備和複雜設置。TDPT 用深度學習 AI 從普通攝影機即時追蹤 24 個身體關鍵點，輸出 3D 座標，透過 VMC Protocol 串流到 Blender/Unity。

⚠️ 傳統即時動捕

  • 昂貴設備 — Vicon/OptiTrack 系統 $50,000+ 起
  • 專用空間 — 需要架設多攝影機陣列的專用場地
  • 穿戴設備 — 演員需穿標記點服裝或感應器
  • 設置時間 — 每次使用前需校準 30+ 分鐘

✅ TDPT 方案

  • 單攝影機即時 — 普通 Webcam/手機攝影機即可，零額外硬體
  • 24 關鍵點 — 深度學習偵測全身主要關節即時輸出 3D 座標
  • VMC Protocol — 即時串流到 Blender/Unity/VSeeFace 等軟體
  • 零設置 — 打開 App 就能用，無需校準或穿戴

攝影機需求

使用單一 RGB 攝影機進行即時骨架偵測，以深度學習模型推斷 3D 空間座標。支援 VMC Protocol 將骨架資料即時串流至 Blender、Unity 等外部軟體。

支援平台與輸出

支援平台
  • Windows — 完整桌面版本
  • macOS — 原生支援 Apple Silicon
  • iOS — 行動裝置版本，利用 ARKit 增強
輸出協定
  • VMC Protocol — 標準虛擬動捕協定
  • OSC — 開放音控協定傳輸
  • Blender / Unity — 透過 VMC 即時接收

優勢分析

  • 即時低延遲 — 適合直播、互動表演、即時預覽等場景
  • 單攝影機即可 — 不需要額外硬體投資，任何 RGB 攝影機都能使用
  • 跨平台支援 — Windows / macOS / iOS 全平台覆蓋
  • 標準輸出協定 — VMC Protocol 被廣泛支援，可接入多種 3D 軟體
  • 操作簡單 — 安裝即用，無需複雜環境設定或校正流程

限制與缺點

  • 僅 24 個關鍵點 — 不包含手指與臉部細節，無法捕捉精細動作
  • 無直接 BVH 輸出 — 需要額外工具串接才能轉換為 BVH 格式
  • 付費軟體 — 非免費方案，需要購買授權
  • 單人追蹤 — 主要設計為單一使用者即時追蹤
  • 精度限制 — 單攝影機深度估計精度有限，不適合正式動畫產出

Pipeline 整合分析

本工具在 AI 動作量產工作流中的定位與搭配方式。

  • 輸入來源 — 攝影機即時串流 / 影片檔案（MP4/AVI）
  • 輸出格式 — VMC Protocol（OSC 即時串流），無直接檔案輸出
  • 上游工具 — 無（TDPT 是管線起點，直接從攝影機取得）
  • 下游工具 — Blender（VMC4B 外掛）→ ARP Remap → FBX/BVH 匯出；VSeeFace/VMagicMirror
  • 推薦搭配 — TDPT → VMC4B(Blender) → ARP Remap 重定向 → Game Export。適合快速參考錄製，非精密動捕。

Pipeline 中的角色

TDPT 是優秀的即時預覽工具，適合在動作製作早期快速驗證方向。但由於關鍵點數量限制（24 點、無手指臉部）及缺乏直接 BVH 輸出，不適合作為正式動畫產出工具。在 AI 動作生成工具 Pipeline 中定位為「前期即時預覽」角色。
