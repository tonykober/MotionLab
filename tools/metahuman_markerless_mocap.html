一、工具概述

MetaHuman Animator Markerless Motion Capture Plugin 是 Epic Games 在 Unreal Engine 5.8 中推出的實驗性功能，將 Meshcapade/SMPL 技術直接整合到引擎內。從單一攝影機影片生成全身+臉部+手指動畫，直接輸出到 MetaHuman 骨架，無需任何外部工具或 retarget。

核心能力一句話：單鏡頭影片 → UE5.8 本地處理 → 全身+臉部+手指動畫 → 直接驅動 MetaHuman → 免費。

解決的問題：

  • 以往需要購買第三方 AI Mocap 服務（Viggle/DeepMotion/Move.AI）
  • 動捕結果需要 retarget 到遊戲骨架（額外步驟+可能品質損失）
  • 臉部和身體需要分別用不同工具追蹤再合併
  • 本功能免費+內建+無需 retarget = 對第三方 Video-to-Mocap 工具的直接競爭

四、使用流程

  1. 安裝 UE 5.8 + 啟用 MetaHuman Animator Plugin
  2. 從 Fab 安裝 MetaHuman Animator Markerless Motion Capture Plugin
  3. 錄製影片 — 用 Webcam/手機/攝影機拍攝表演（遵循 Performance Capture Guidelines）
  4. 匯入影片 — 透過 Live Link Hub 的 Capture Manager 建立 Capture Data Asset
  5. 設定解算 — 在 MetaHuman Performance Editor 中設定 solve settings（face/body/both）
  6. 執行 Solve — 本地批次處理影片
  7. 匯出 Animation Sequence — 標準 UE Animation Sequence
  8. 套用到 MetaHuman — 在 Level Sequence 中指派動畫到角色（無需 retarget）

六、優缺點分析

✅ 優點

  1. 完全免費 — 無訂閱、無按量計費、無限使用
  2. 引擎內建 — 無需外部工具，無需匯入/匯出/retarget
  3. 全身+臉+手一體 — 單一工具解決所有追蹤
  4. 本地處理 — 離線、隱私、不依賴雲端
  5. MetaHuman 直出 — 所有 MetaHuman 共用骨架，無需 retarget
  6. 基於 Meshcapade/SMPL — 頂級學術技術背景
  7. 雙鏡頭模式 — 可分離臉部和身體追蹤提高品質

❌ 缺點

  1. 僅 Windows — macOS/Linux 不支援
  2. 實驗性 — 可能有 bug、可能未來版本變更
  3. 僅 MetaHuman 骨架 — 不能直接用於非 MetaHuman 角色（但 MetaHuman 骨架可 retarget）
  4. 非即時 — 批次處理（不像 Marionette 可即時串流）
  5. 僅 UE 生態系 — Unity/Godot 用戶無法使用
  6. 品質未知 — 剛發佈，尚無大量用戶品質反饋
  7. 需要 UE 5.8 — 較新版本，現有項目可能還在 5.3/5.4
